mqrestt
===============

MQtt REST Translator, a MQTT<->REST bridge dameon


ROADMAP
=======
0.1:
    - MQTT->REST support
    - REST->MQTT support

0.2
    - SSL support both MQTT and HTTPs
    - rewrite support, to rewrite MQTT topics into URL's and vica verse (a'la Apache mod_rewrite)
